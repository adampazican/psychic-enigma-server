# Psychic enigma server

Nodejs server api for teachers to post materials online. Main purpose of this project is to experiment with nodejs and react.

## Dependencies

- nodejs 8
- mongodb
- bcrypt
- body-parser
- dotenv
- express
- jsonwebtoken
- mongoose
- mongoose-type-url
- passport
- passport-jwt
- .env variables DB_HOST, JWT_SECRET, SALT_WORK_FACTOR
